(static) module:extras/gearmesh.createGear(innerRadius, outerRadius, thickness, teeth, toothDepth)

Builds a mesh buffer representing a gear centered at the origin.

Parameters

  • innerRadius (Type: number)
    Inner radius of the gear wheel
  • outerRadius (Type: number)
    Outer radius of the gear wheel, at the valleys between teeth.
  • thickness (Type: number)
    Thickness of the gear
  • teeth (Type: number)
    Number of teeth.
  • toothDepth (Type: number)
    Depth of each gear tooth.

Return Value

Return value. (Type: MeshBuffer)
